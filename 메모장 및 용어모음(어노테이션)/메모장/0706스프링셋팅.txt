%스프링 부트 3.0 의 5가지의 규칙%
1.인터페이스를 이용하여 의존성을 만든다.
2.인스턴스를 명시적으로 생성하지 않는다.
3.어노테이션을 클래스에 부여한다.
4.스프링 프레임 워크에서 인스턴스를 생성한다.
5.인스턴스를 이용하고 싶은 곳에 어노테이션을 부여한다.

인텔리제이 스프링 부트 셋팅 방법
1. 스프링 부트 포터블 파일 실행한다(설치할 때 파일경로 맞게 설치)
2. Spring Initializr 사이트 들어가서 SpringBootDev를 설치한다
&&스프링 부트 버전 3.0.8
&&  Java 버전은 17
&&  ADD Dpendency Injection 해준다
&&  SpringBootDev 제너레이트 해줌 - 다운로드 됨
3.프로젝트 담을 파일을 하나 만들어줌
4.포터블 버전은 Do not Import 해준다 뉴 프로젝트 만들어서 테스트 파일 생성되나 한번 확인해보고
5. New Project 에서 Location을 해당 위치에다 설치를 해준다 %%JDK 버전 확인%% 17
6. Spring Initializr 에서 다운받은 SpringBootDev는 Project 안에 같이 보관해준다
7. SpringBootDev 에서 gradle을 적용시킬려면 Project 밑에 test랑 같이 보관해주고
8. 파일을 한번 불러온다
